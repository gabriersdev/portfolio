## Overview

A portal for consulting allied professionals within the community, developed in PHP with Bootstrap and TailwindCSS.

## Technologies Used

-   **Code**: PHP e JS
-   **Database**: MySQL
-   **Frameworks:** JQuery, JQuery Mask, JQuery UI, JQuery Autocomplete e Sweet Alert
-   **Styling:** Tailwind CSS e Bootstrap
